ActiveMQ Port already in use diyebilir ve gözükmeyebilir.

Bunun için sırasıyla.


netsh interface ipv4 show excludedportrange protocol=tcp

ile TCP protokülünün rezerve portlarını görüyoruz. Sonra CMD yada PowerShell yönetici olarak
çalıştırıyoruz.


1. net stop winnat
2. netsh interface ipv4 set dynamic tcp start=61518 num=1000
3. net start winnat

diyerek işlemi bitiriyoruz.
